# User Authentication System

Secure your **MERN Stack App** with **JWT-Based User Authentication and Authorization**

---
## Features
✅ User Signup  
✅ User Login  
✅ Password Encryption with **bcrypt**  
✅ JWT-based Authentication  
✅ Protected Routes  
✅ Cookie-based Session Handling

---

## Technologies 

### **Frontend**
- React.js
- React Router
- Axios
- React Toastify
- React Cookie

### **Backend**
- Node.js
- Express.js
- MongoDB 
- Bcrypt
- JWT (jsonwebtoken)
- Mongoose
- Dotenv
- CORS
- Cookie-Parser
